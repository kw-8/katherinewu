Portfolio

Instagran | Repo

Instagran is a grandparent-themed Instagram. Featuring an explore page, user profile pages, and a custom-made React carousel.

I built Instagran in a couple weeks using Ruby on Rails, AWS S3, and React-Redux. In those couple weeks, I researched and implemented AWS storage, jBuilder, and became well-acquainted with React-Redux reducers and state.

NextGlass | Repo

Recommendations from your favorites! NextGlass is a web application that uses your wine preferences to suggest new wines to taste.

From finding the dataset to creating the suggestions algorithm to polishing up the collection and adding pagination, I have touched every part of this MERN stack. As team lead, I learned to truly value frequent check-ins and committing to the vision.

Bubblepedia | Repo

Bubblepedia is a bubble tea themed Wikipedia interface that allows you to explore new articles in a fun way. The main feature is the related article bubbles that appear when clicking the 'See Also' button. These appear for every article with a Related Articles section.

Bubblepedia was built using plain JavaScript, the Wikipedia article and search APIs, and GreenSock (GSAP) for the bubble movements.
